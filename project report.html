Features Implemented
  1. Header Navigation: Links to Home, Food, Recipe, and Project Report pages.
  2. Responsive Layout: The page adapts to different screen sizes.
  3. Recipe Cards: Each recipe has an image/icon, a title, and a short description.
  4. Search Bar: Users can filter recipes in real-time based on input keywords.
  5. Clean UI: Use of colors, spacing, and hover effects for better user experience.
  6. Hosting on GitHub Pages: Accessible live via [Food Recipe Webpage] "https://sonofgrace2203.github.io/Food-Recipe".
Challenges Faced
  1. Ensuring the webpage remained responsive across all screen sizes.
  2. Implementing the search functionality to work with both titles and descriptions.
  3. Managing GitHub deployment and resolving CSS file path issues.
Lessons Learned
  1. How to structure a webpage semantically with HTML
  2. The importance of CSS responsiveness using grid and flexbox.
  3. How to implement DOM manipulation in JavaScript.
  4. Team collaboration and version control with GitHub.
